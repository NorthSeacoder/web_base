# 时间复杂度

T(n)=O(ƒ(n))

TIP

T(n):算法中所有语句频度之和
频度:一个语句的频度是指该语句在算法中被重复执行的次数
O(时间复杂度):T(n)的数量级
一般取ƒ(n)中随 n 增长最快的项将其系数设置为 1 作为时间复杂度的度量,例如ƒ(n)=an3+bn2+cn+d,其时间复杂度为 O(n3)
